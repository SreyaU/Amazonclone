sdcard module to use with stm32f103

Using an SD card module with the STM32F103 microcontroller involves several steps, including hardware connections, configuring the software libraries, and writing the necessary code to read from and write to the SD card. Here’s a step-by-step guide:

Hardware Connections

  1. SD Card Module Pins:

    • VCC: Connect to 3.3V or 5V (depending on your SD card module's voltage requirement).
    • GND: Connect to Ground.
    • CS (Chip Select): Connect to an available GPIO pin (e.g., PA4).
    • MOSI (Master Out Slave In): Connect to the SPI MOSI pin (e.g., PA7).
    • MISO (Master In Slave Out): Connect to the SPI MISO pin (e.g., PA6).
    • SCK (Serial Clock): Connect to the SPI SCK pin (e.g., PA5).

  2. STM32F103 SPI Pins:

    • PA4: Chip Select (CS)
    • PA5: Serial Clock (SCK)
    • PA6: Master In Slave Out (MISO)
    • PA7: Master Out Slave In (MOSI)

Software Configuration

  1. CubeMX Configuration:

    • Open STM32CubeMX and create a new project for the STM32F103.
    • Configure the SPI peripheral in Full-Duplex Master mode.
    • Set the GPIO pins for the SPI (SCK, MISO, MOSI) and the Chip Select (CS) pin.
    • Configure the clock settings according to your needs.
    • Generate the initialization code and open it in your preferred IDE (e.g., STM32CubeIDE).

  2. FatFS Library:

    • STM32CubeMX can automatically integrate the FatFS file system library.
    • Enable the FatFS middleware in STM32CubeMX.
    • Configure the FatFS library to use SPI for SD card communication.

Code Implementation

  1. Include Necessary Headers:

    c
    #include "fatfs.h"
#include "spi.h"

  2. Mount the SD Card:

    c
    FATFS fs;
FRESULT res;

res = f_mount(&fs, "", 1);
if (res != FR_OK) {
    // Handle error
}

  3. Open/Create a File:

    c
    FIL file;
res = f_open(&file, "test.txt", FA_CREATE_ALWAYS | FA_WRITE);
if (res != FR_OK) {
    // Handle error
}

  4. Write to the File:

    c
    char data[] = "Hello, STM32!";
UINT bytes_written;
res = f_write(&file, data, sizeof(data), &bytes_written);
if (res != FR_OK || bytes_written != sizeof(data)) {
    // Handle error
}

  5. Close the File:

    c
    res = f_close(&file);
if (res != FR_OK) {
    // Handle error
}

  6. Unmount the SD Card:

    c
    f_mount(NULL, "", 1);
